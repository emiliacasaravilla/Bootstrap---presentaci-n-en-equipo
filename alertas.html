La idea es poder usar estilos que ayuden al usuario a identificar qué tipo de alerta es. ¿Y esta amarilla? Advertencia (warning). Además vamos a aprovechar para mostrar que también se puede agregar links a las alertas y que el color de la letra del link combine con el del fondo de la alerta. ¿Cómo? Acá te mostramos.

Esta podría haber sido una simple alerta de información (info) pero la usamos para mostrar que las alertas también pueden contener elementos HTML adicionales como encabezados, párrafos y divisores.
