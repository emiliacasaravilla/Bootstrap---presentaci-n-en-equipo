Tarjetas de Bootstrap

Esto es una tarjeta básica de Bootstrap. Como podrás notar, no tiene márgenes ni rellenos de espaciado, así que para utilizarla por sí sola, su visualización no es muy linda. Vamos a utilizar la clase card dentro de un <div> 

<div class="card">
        <p class="mb-0">Esto es una tarjeta básica de Bootstrap. Como podrás notar, no tiene márgenes 
            ni rellenos de espaciado, así que para utilizarla por sí sola,
            su visualización no es muy linda. Vamos a utilizar la clase <b>card</b> dentro de un
            <b>&lt;div&gt;</b>
        </p>
</div>

Normalmente utilizaremos las tarjetas con un cuerpo de tarjeta de la clase card-body, así se ve el texto dentro del cuerpo de una tarjeta. Si no se especifíca un ancho, el mismo toma el tamaño que necesite para mostrar el contenido. 

<div class="card">
        <div class="card-body">
            <p class="mb-0 card-text">Normalmente utilizaremos las tarjetas con un cuerpo de tarjeta de
            la clase <b>card-body</b>, así se ve el texto dentro del cuerpo de una tarjeta.
            Si no se especifíca un ancho, el mismo toma el tamaño que necesite para mostrar el contenido.
            </p>
        </div>
</div>
Título de tarjeta
Esto es un subtítulo

Las tarjetas también suelen llevar un título y enlaces o botones, también notarás que el tamaño de esta tarjeta es mucho más pequeño, para esto modificamos el width en un atributo style.

Documentación tarjetas 
<div class="card" style="width: 18rem;">
        <div class="card-body">
            <h5 class="card-title">Título de tarjeta</h5>
            <h6 class="card-subtitle mb-2 text-muted">Esto es un subtítulo</h6>
            <p class="card-text">Las tarjetas también suelen llevar un título y enlaces 
            o botones, también notarás que el tamaño de esta tarjeta es mucho más pequeño,
            para esto modificamos el <b>width</b> en un atributo <b>style</b>.</p>
            <a href="https://getbootstrap.com/docs/5.0/components/card/"
              class="btn btn-primary d-flex justify-content-center card-link">
              Documentación tarjetas
            </a>
        </div>
</div>
