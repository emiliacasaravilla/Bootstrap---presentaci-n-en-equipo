Navbar

Es una barra de navegación que contiene links a otras secciones o páginas, input de búsqueda, menú de usuario, etc.

Contenido

Este componente cuenta con varias formas de personalizarlo, además de algunas utilidades para facilitar su uso, como la capacidad de agrupar los links en un menú colapsable en pantallas chicas.

Primero veamos el caso más simple, una barra de navegación con links:

Navbar
Home Modal Dropdown Deshabilitado 
<nav class="navbar navbar-expand-sm navbar-light bg-light rounded">
	<div class="container-fluid justify-content-start">
		<a class="navbar-brand" href="#">Navbar</a>
	
		<div class="navbar-nav">
			<a class="nav-link active" aria-current="page" href="#">Home</a>
			<a class="nav-link" href="#">Modal</a>
			<a class="nav-link" href="#">Dropdown</a>
			<a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Deshabilitado</a>
		</div>
	</div>
</nav>

Colapsable

Luego, podemos hacer que se muestre el menú de navegación en dispositivos móviles. Para ello, vamos a agregar el botón del menú: 

<button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavAltMarkup" aria-controls="navbarNavAltMarkup" aria-expanded="false" aria-label="Toggle navigation">
	<span class="navbar-toggler-icon"></span>
</button>

Luego, agregamos un div por fuera del div con clase navbar-nav, de la siguiente manera: 

<div class="collapse navbar-collapse" id="navbarNavAltMarkup">
	<div class="navbar-nav">
		....
	</div>
</div>

OJO: Debe coincidir la id del div que recién agregamos con el atributo data-bs-target del botón.

El resultado es el siguiente

Navbar
  • Home
  • Modal
  • Dropdown
  • Deshabilitado 
<nav class="navbar navbar-expand-lg navbar-light bg-light">
	<div class="container-fluid">
		<a class="navbar-brand" href="#">Navbar</a>
		<button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavAltMarkup" aria-controls="navbarNavAltMarkup" aria-expanded="false" aria-label="Toggle navigation">
			<span class="navbar-toggler-icon"></span>
		</button>
		<div class="collapse navbar-collapse" id="navbarNavAltMarkup">
			<div class="navbar-nav">
				<a class="nav-link active" aria-current="page" href="#">Home</a>
				<a class="nav-link" href="#">Modal</a>
				<a class="nav-link" href="#">Dropdown</a>
				<a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Deshabilitado</a>
			</div>
		</div>
	</div>
</nav>

Marca

Para agregar una marca, podemos utilizar la clase .navbar-brand en un elemento <a> o <span>.

Búsqueda

También podríamos implementar un input de búsqueda! Para esto, vamos a utilizar la clase .form-inline en el elemento <form> y la clase .form-control en el elemento <input>.

Navbar 
<nav class="navbar navbar-light bg-light">
	<form class="form-inline">
		<input class="form-control mr-sm-2" type="search" placeholder="Search" aria-label="Search">
		<button class="btn btn-outline-success my-2 my-sm-0" type="submit">Search</button>
	</form>
</nav>

Colores

Para cambiar el color de fondo de el navbar, podemos utilizar las clases .navbar-light si usamos un fondo claro, o .navbar-dark si usamos un fondo oscuro.
Luego, usamos las clases bg-* para cambiar el color de fondo.
